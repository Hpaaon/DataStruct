# Linear Structures

## Queue 队列 (FIFO)

>队列是一种限制性线性表，它强制执行“排队”逻辑：数据从一端进入（Rear），从另一端离开（Front）。

### 链式队列 （Linked Queue）

>用链表实现队列，优势在于其**动态性**。

### 循环队列 （Circular Queue）

>与链式队列相比，它最大的优势在于**内存的复用**：它预先分配一块连续内存，通过索引取模运算让逻辑首尾相连，避免了频繁调用 new/delete 带来的开销和内存碎片。

### 双端队列 （Double-ended Queue）

>它打破了“先进先出”或“后进先出”的单一限制，允许你在序列的两端以 $O(1)$ 的时间复杂度进行插入和删除。

### 优先队列 （Priority Queue）

>普通队列讲究“先来后到”，而优先队列讲究**能者优先**。

在优先队列中，每个元素都被赋予了一个**优先级**。

* 出队规则：不是出的最早进来的那个，而是出当前队列中优先级最高（或者是数值最大/最小）的那个。

* 入队规则：随便进，但进去后它会自动排到它该去的位置。
